# Hypsteria
A green-ish low constrast retro' color scheme for VIM inspired by gruvbox, 
darcula and solarized.
<br>
<br>

### Installation <br>
<hr>

#### Manual:
- clone the repo and put the .vim file in the `~/.vim/colors` folder

#### VIM Plug:
- Add
`Plug 'BrainDeath0/Hypsteria'`
to your .vimrc

<br>
<hr>

#### Tips:
Enjoy with coffee.
